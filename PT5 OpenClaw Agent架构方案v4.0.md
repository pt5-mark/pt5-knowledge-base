## 关键配置要点

### 知识库同步机制
- **同步方式**: git + webhook
- **存储**: GitHub私有仓库
- **Agent访问**: REST API接口
- **实时更新**: webhook触发

### 权限控制
- **CEO Agent**: 读写全部目录
- **Manager Agent**: 读写本部门 + 读报表
- **Worker Agent**: 读共享模板 + 本领域知识

### 定时任务
- CEO Agent: 每天8点生成日报
- Manager Agent: 每天18点生成部门报告
- Worker Agent: 实时处理任务

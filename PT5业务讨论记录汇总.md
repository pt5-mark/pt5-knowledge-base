## 一、PT5 Global 旅游业务着陆页重构

### 1.1 项目背景

**公司名称**: PT5 Global (香港柏盛中国外商代理有限公司)  
**目标市场**: 外国游客入境中国旅游市场  
**核心痛点**: 外国人到中国面临语言、支付、导航等多重障碍

### 1.2 重构方案 (AIDA模型)

#### 核心理念
**"不是推销会员，而是展示独创性服务价值，让客人主动选择"**

#### 页面结构 (7大板块)

```
1. HERO 首屏 - Attention
   - 独创性价值主张: "不是旅游，是回家"
   - 互动国家选择器
   - CTA: 探索我的中国之旅

2. 签证&航班资讯 - Interest
   - 根据选择的国家自动展示
   - 签证政策 (免签/落地签/需申请)
   - 推荐航班 & 口岸
   - FAQ专区

3. 旅游陪同服务 - Desire
   - **独创"Mom Guide"概念**: 本地中国向导像妈妈一样照顾外国游客
   - 真实场景: 家庭晚餐、早市采购、乡村体验
   - 价格: $400/天 (8小时)
   - 包含: 翻译、导航、预订、应急处理

4. 旅游采购服务 - Desire
   - 1688/工厂源头直采
   - 真人验货 (Mom Guide亲自检查)
   - 服务费: 采购金额的15%
   - 物流: 空运7-10天，海运30-45天

5. 数字化生存服务 - Desire
   - **预装手机方案**: 到手即用的中国App手机
   - 包含: 支付宝(已实名)、微信、滴滴、高德地图
   - 买断: $100 | 租赁: $8/天
   - 5G流量卡 (7天/15天/30天)

6. 注册与会员 - Action
   - **免费注册**: 获取旅行指南PDF，加入咨询群
   - **预付会员**: $200 → 终身8折 + WhatsApp终身服务 + $210价值包
   - **自动升级**: 累计服务费满$200自动成为会员

7. Footer
   - 联系方式、社交媒体、法律信息
```

### 1.3 关键设计决策

| 决策 | 说明 |
|------|------|
| **NOT pushy** | 不强制推销会员，先展示服务价值 |
| **互动式** | 国家选择器作为入口，个性化内容 |
| **社交证明** | 真实客户评价，真实Mom Guide照片 |
| **透明定价** | 所有服务价格明确，无隐藏费用 |
| **FAQ全覆盖** | 每个板块都有专属FAQ消除顾虑 |

### 1.4 会员体系设计

**三种转化路径**:

| 路径 | 门槛 | 权益 |
|------|------|------|
| 免费注册 | 0 | 基础信息、咨询群 |
| 累计升级 | 服务费满$200 | 自动获得8折+WhatsApp终身 |
| 预付会员 | $200 | 立即获得8折+WhatsApp+$210价值包 |

**$210价值包包含**:
- 数字生存包 ($80)
- 机场接机 ($30)
- 家庭晚餐 ($50)
- 1天在线顾问 ($50)

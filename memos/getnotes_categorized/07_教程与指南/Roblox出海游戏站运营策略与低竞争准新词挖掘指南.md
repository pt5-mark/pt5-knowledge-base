🎯 **核心战略：游击战思维**
- **避实击虚**：弱小阶段避免与高权重IO老站直接竞争，转向巨头未关注的薄弱领域（如Roblox工具站）
- **集中突破**：将资源聚焦于单一低竞争领域，实现”低竞争状态下的搜索套利”
- **案例佐证**：Raise Animals网站通过价值计算器、天气价值计算器等工具，2周内实现日活近万

⚠️ **新手误区与失败案例**
- **盲目追热**：看到Google Trends飙升词便仓促建站（如”sprunki”游戏），90%新站沦为炮灰
- **错误操作链**：购买域名→搭建网站→挂游戏→坐等流量，忽视搜索意图与竞争分析
- **竞争劣势**：新站缺乏老站数千外链、数百篇内容及长期域名权重积累，难以突破SERP前10页

🔍 **低竞争”准新词”四大特征**
1. **萌芽阶段**：已在核心YouTuber圈子讨论，具备热度基础
2. **未破圈**：大众玩家未知晓，主流媒体未报道，Google Trends刚出现波动
3. **SERP干净**：搜索结果多为论坛帖子/个人主页，缺乏专业内容站
4. **域名可注册**：相关IO后缀域名尚未被注册

🚀 **N8N+AI挖词工作流搭建**
1. **准备工具**
   - N8N自动化平台（Docker本地部署免费）
   - YouTube API Key与Google Cloud OAuth授权
   - OpenAI/Gemini API（用于视频内容分析）
   - Google Sheets（数据存储与管理）

**三大核心工作流**

- **网红数据库初始化**：通过YouTube API抓取订阅博主信息，自动存入表格

- **24小时视频监控**：定时检查博主新视频，AI提取游戏名称（如从标题”THIS NEW ROBLOX GAME WILL BLOW YOUR MIND!!“识别具体游戏）

- **AI聚类分析**：计算”热度分=提及视频数×10+总播放量/1000+百万粉博主提及数×50”，生成Top3游戏报告

📊 **人工验证四步法**
1. **趋势判断**：Google Trends出现明显上升曲线
2. **竞争分析**：搜索结果首页无权威内容站，IO域名未被注册
3. **玩法评估**：通过AI识别游戏核心痛点，判断工具/内容开发潜力
4. **用户数据验证**：Steam/Roblox关注点赞收藏量，Itch查看用户评论需求

💡 **关键洞察**
- 新站优势：当搜索需求出现而优质内容空缺时，Google会优先推送新站填补空白
- 时机窗口：准新词从出现到破圈周期约7-14天，需快速完成内容部署
- 差异化生存：要么提供独家游戏资源，要么拥有强外链，或转向TikTok/Facebook付费引流

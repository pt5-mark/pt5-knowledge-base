# 兼职团队费用管理细则

> 生效日期：2024年7月15日
> 创建人：mark4all
> 来源：桌面文档导入

---

## 背景

为了进一步对兼职队伍的管理规范，在此对兼职队伍的各项费用进行声明，包括了周任务奖励、业务提成和团队提成三个方面。

---

## 一、周任务

**参与条件：** 完成学习，通过作业后即可参与

### 模式
- 任务基础分为**必做**与**选做**
- 完成基础部分：领取 **200元** 周任务奖励
- 月内持续四次完成：额外奖励 **200元**
- 多做任务：最多每周可领取 **400元**

### 发放流程
```
系统领取 → 完成 → 系统提交 → 公司验收 → 财务核算 → 发放
```

- **时间**：当月任务，公司当月完成审核，审核通过后次月统一发放
- **方式**：个人银行卡（需要提交银行卡卡号）

---

## 二、跟单提成

> 自主报名，公司分配，视频指导，边学边做

| 毛利区间 | 提成比例 | 说明 |
|---------|---------|------|
| 10,000元以内 | **8%** | 由任务跟单人员享有 |
| 10,000元以上 | **12%** | 由任务跟单人员享有 |

**特殊政策：**
- 跟单毛利**不计入**开发提成的当月总毛利
- 跟单毛利当月超过 **3,000元**，豁免当月每周任务
- 跟单客户（每个客户当月至少两单）达到 **6人**，豁免当月每周任务

---

## 三、开发提成

> 自主获客，高额提成

### 梯度提成表

| 当月毛利 | 提成比例 | 额外奖励 |
|---------|---------|---------|
| 2,000元以上 | - | **+200元** |
| 10,000元以下 | **15%** | - |
| 10,000-30,000元 | **20%** | - |
| 30,000元以上 | **25%** | - |
| 50,000元以上 | **30%** | - |
| 100,000元以上 | **35%** | - |

**新客政策：**
- 以上新客均可增加 **5%** 额外提成
- 自主开发客户从第一次下单开始，**六个月内**成交，不计梯度额外提成 **5%**

### 开发客户定义
- 通过公司平台上架产品获得
- 借由公司开发工具、兼职平台、社媒平台等个人渠道方式参与开发
- 可寻求团队长或者 Mark 帮助成单并跟单获得的客户

---

## 四、团队提成

> 奖励在新人业务过程中予以新人帮助的团队长

### 1. 团队长资质

- [x] 具有日常操作店铺，在客户群和与客人日常交流的成功经验
- [x] 至少有一个常规出单的客户在手操作
- [x] 最高当月毛利（含开发和跟单）达到过 **10,000元以上**
- [x] 乐于相互合作，传授经验

### 2. 团队长职能

- 巩固新人的业务知识、业务技能，解答新人业务操作中产生的疑难
- 熟练使用录屏软件指导新人业务（新业务需要有完整的视频指导操作）
- 对新人业务操作过程中各个流程进行监督，避免业务犯错造成公司受损失

### 3. 团队长权限

- 发布报价任务和跟单任务
- 对流失队员订单可以二次指派，给有潜力的新人
- 队员开除：拒不配合、缺乏专业性的新人可以报备公司踢出兼职队伍

### 4. 团队长津贴

| 津贴类型 | 条件 | 金额 |
|---------|------|------|
| **开单津贴** | 团队成员首次当月毛利≥2,000元 | **100元/人**（一新人一次） |
| **基本津贴** | - | 本团队总毛利的 **2%** |
| **成长津贴** | 团队成员当月毛利≥5,000元 | 该成员毛利的 **3%**（多次） |
| **育成津贴** | 团队成员当月开发毛利≥10,000元，次月自动脱离团队 | 该成员今后业务当月毛利超过1万以上的，按全部毛利 **3%** 提成 |
| **荣誉津贴** | 当月团队成员人均毛利达到2,000元，且总毛利达到10,000元（不含本人） | **500元** |

## 五、附文补充

### 1. 产品管理
- 所有新产品，由运营审核产品后发布
- 另需提交1688产品链接备查
- 产品发布人需要保证页面资料后续的更新和维护

### 2. 毛利计算方法
```
毛利 = 客户收入 × 执行汇率 - 银行成本 - 产品成本 - 运费成本 - 包装成本 - 退赔成本
```

### 3. 客户属性界定
符合下述情况的客人带来的订单，可以进行提成：
- 有在 WhatsApp、Skype、微信拉群，或在带教群领取任务的
- 订单详情在公司系统和相关表格登记的

**特殊情况：** 个人自带客户并成交的，可以提前报备公司，与公司商定提成和业务操作方式，此外这部分业务不纳入常规业务核算。

### 4. 亏损处理

| 情况 | 处理方式 |
|------|---------|
| **经公司授权**进行的业务 | 公司承担亏损，业务员无提成 |
| **未经公司授权**，自行决定关于价格、时效、运输渠道、信用额度等相关事宜，并引起的亏损 | 由个人承担，公司保留追索的权利 |

### 5. 结算规则
- 业务收入结算以**收款和发货**作为结项
- 跨月未发货或未收款业务递归到次月
- 如遇退货和索赔的，可向前追溯作为成本扣除

### 6. 客户管理
- 客户投诉或明确告知要求更换代理的，公司有权再次进行客户分配
- 兼职期间开发的客户都属于**公司所有**，属于公司资产，需要将客户信息进行登记
- 所有客户和业务数据需要按要求记录在**公司系统**，作为费用发放和提成的依据

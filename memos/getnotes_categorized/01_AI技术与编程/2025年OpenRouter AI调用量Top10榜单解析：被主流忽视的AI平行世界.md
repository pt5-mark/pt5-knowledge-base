💡 **典型案例深度解析**

### 1. Kilo Code

**定位**：开源+即装即用的VS Code AI coding agent

**核心能力**：

自动处理依赖管理、bug定位、文档更新等琐碎任务

Orchestrator模式：拆解复杂项目为子任务，分配给Architect/Code/Debug等角色agent

支持400+模型，无需API Key配置，提供$20免费额度

**数据**：GitHub近万星，VS Code安装量超29万

### 2. Cline

**差异化**：”自主但可控”的开发流程，执行前需用户确认每步操作

**生态兼容**：支持OpenRouter/Anthropic/本地模型（Ollama等），无模型锁定

**市场表现**：GitHub 5万+星，日活用户超百万，A轮融资3200万美元

### 3. SillyTavern

**定位**：高级玩家的本地LLM前端工具

**特色功能**：

角色卡（Character Cards）预设对话风格

支持RAG文档嵌入、图像生成（Stable Diffusion）和web搜索

完全开源，无外部融资

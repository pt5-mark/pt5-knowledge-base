🔧 **整体实现思路**

1. 火山引擎注册账号并实名认证

2. 申请即梦4.0 API Key（与即梦交互的“钥匙”）

3. n8n搭建工作流：手动触发 → 调用即梦API → 解析结果 → 下载图片 → 保存本地

📌 **关键步骤详解**

### 1. 开通即梦4.0服务

火山引擎官网：[https://www.volcengine.com](https://www.volcengine.com)

路径：产品 → 人工智能与机器学习 → 图像生成大模型 → 立即体验

实名认证：个人认证支持微信扫码+人脸识别（10秒完成），1个身份可认证3个账号

开通服务：选择“即梦AI-图片生成4.0”，免费试用含200张生成额度，正式付费0.22元/张

### 2. 获取API Key

火山方舟（火山引擎旗下大模型平台）路径：产品 → 人工智能与机器学习 → 火山方舟 → 控制台 → API Key管理 → 创建Key

API Key用于n8n调用接口时的身份验证

### 3. n8n工作流搭建

**触发节点**：When clicking ‘Execute workflow’（手动触发）

**HTTP请求节点（调用API）**

Method: POST

URL: [https://ark.cn-beijing.volces.com/api/v3/images/generations](https://ark.cn-beijing.volces.com/api/v3/images/generations)

请求头：Authorization: Bearer $ARK_API_KEY

请求体示例：包含model（doubao-seedream-4-0-250828）、prompt（详细提示词）、size（2K）等参数

**Code节点（解析结果）**：提取返回的图片URL列表

**HTTP请求节点（下载图片）**：使用解析后的URL请求图片

**文件读写节点**：将图片保存到本地（注意Docker路径映射）

🎯 **案例效果**

- 生成图片示例：4张室内私房摄影风格图片（25岁亚洲女性，黑色蕾丝服饰，红金灯光氛围）

- 图片命名格式：202511111081847651.png等

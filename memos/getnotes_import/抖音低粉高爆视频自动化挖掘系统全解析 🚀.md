🛠️ **n8n自动化工作流详解**

### 1. 数据去重环节

- **节点配置**：查询飞书→飞书收集

**核心代码**：
python
detail_data = _input.first().json.data["items"]
title_list = []
for desc in detail_data:
if len(desc.fields) == 0:
    continue
desc = desc.fields.title[0].text
title_list.append(desc)
output.append({"ret":title_list})
- **功能**：从飞书多维表格读取已收录标题，避免重复采集

### 2. 视频搜索环节

- **工具**：TikHub API（支持抖音/小红书/B站等多平台数据接口）

**关键参数**：

- sort_type: 1（按热度排序）

- publish_time: 1（最近1天发布）

- filter_duration: 0-1（1分钟以内视频）

- **数据输出**：包含标题/封面/作者/粉丝数/互动数据的完整视频信息

### 3. 智能筛选环节

- **流程节点**：Loop Over Items→飞书是否已有→Code→是否低粉高爆

**双重校验**：

- 飞书查重（避免重复收录）

- 数据阈值判断（点赞/评论/粉丝数三维筛选）

### 4. 内容分析环节

- **处理流程**：视频下载→提取文本→存入本地→写入飞书

- **AI应用**：Gemini AI提取语音转文字（需美国网络环境）

- **成果沉淀**：包含视频链接/文案内容/数据指标的完整素材库
